About me

My name is Neta Elad, I am a CS/Physics undergraduate student in the Technion, Haifa. This project was done under the guidance and mentoring of Joan Adler. For more information about the project, I can be contacted at elad.neta+aviz@gmail.com.

 

The website was generated from markdown files, using the npm package markdown-styles and some hand-crafted helper scripts. The website was created with a modern look and feel, but should work with older browsers, and even with JavaScript (and CSS) disabled. This was tested, though not extensively.

 

All of the examples, visualizations, input files processing (as described in the scripts section) and website generation were done on my personal computer:
